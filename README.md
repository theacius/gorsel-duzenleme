# Görsel Şablon Stüdyosu

**Teklif ve sunum görsellerini şablonla üret.**

Görsel düzenleme aracı; arka plan görseli üzerine alanlar yerleştirerek teklif, fiyat listesi ve benzeri doküman şablonları oluşturmanızı sağlar. Şablonları tarayıcıda doldurup PDF veya Word olarak dışa aktarabilirsiniz.

---

## Özellikler

- Görsel şablon editörü (sürükle-bırak alanlar)
- Canlı önizleme ve doldurma modu
- PDF / Word dışa aktarma
- Şablon içe/dışa aktarma (JSON)
- Doldurma geçmişi
- İsteğe bağlı parola koruması

---

## Teknoloji

| Alan | Teknolojiler |
|------|----------------|
| Ön yüz | Next.js 15, React, TypeScript, Tailwind CSS |
| UI | HeroUI |
| Dışa aktarma | html2canvas, jsPDF, docx |
